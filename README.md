# Stellar Burgers

## Frontend

- Typescript
- CSS Modules
- React + React Router
- Redux + Redux Thunk

## Tests

- Jest
- Cypress

## How to start

- Clone repository – `git clone https://github.com/klimaleks12133/react-burger.git`
- Install the dependencies – `npm i`
- Start the project – `npm start`
- Go to http://localhost:3000/
